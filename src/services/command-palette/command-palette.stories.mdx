## Botões padrões

Usados em geral para componentes angular. Pra mudar a cor, utilizar o input `type`.
As cores são atribuídas por meio do model `MainColors`

Devem ser usados da seguinte forma:

```
<pm-button [type]="'primary'">
  Button Content
</pm-button>
```

## Module Config

```
import {CommandPaletteModule} from '@mprisma/components';

@NgModule({
  imports: [
    ...,
    CommandPaletteModule.forRoot({
        hotkeys: ['meta+k', 'ctrl+k'],
        // pode ser atualizada posteriormente atribuindo ao serviço:
        // this.commandPalette.commandPaletteEntriesService.paletteEntries = [{id: 'a', label: 'A', action: () => {console.log('a')}}]
        paletteEntries: []
    }),
    ...,
  ]
})
```
